# 提示词组装模板 — 快速参考

本文件是 P4 阶段快速参考，包含完整的提示词组装示例。

## 一键组装公式

```
[风格前缀 — 复制粘贴，不改一字]
+
[内容描述 — 根据观点和研究数据定制]
+
[风格后缀 — 复制粘贴，不改一字]
```

## 风格前缀（直接复制）

Notion官方插画风格松弛感线稿信息图。线条特征：纯黄色为主，搭配其他适合的颜色。线条粗细不均匀，像马克笔随手画的质感。笔触松弛、略带抖动，不追求工整。所有图标、人物、图表都保持手绘涂鸦感。少部份地方填充黑色让整体压实。人物画法：简笔画风格女孩，短发，圆润的头、点或线表示五官，肢体夸张、动作松弛自然，可以是完整人物或只露出局部。

## 风格后缀（直接复制）

禁止事项：不要彩色渐变或复杂配色。不要粗黑边框或生硬分隔线。不要3D效果、阴影、立体感。不要密集文字堆砌，文字内容简化。保持大量留白和呼吸感。装饰元素要克制。图片比例16:9，中文标注。
