# Notion 手绘风格 — 完整风格规范

本文件定义了所有信息图共享的视觉风格系统。
每张图的提示词必须包含完整的风格前缀和后缀，不可省略。

## 风格前缀（每张图必须完整包含）

```
Notion官方插画风格松弛感线稿信息图。

线条特征：
- 纯黄色为主，搭配其他适合的颜色
- 线条粗细不均匀，像马克笔随手画的质感
- 笔触松弛、略带抖动，不追求工整
- 所有图标、人物、图表都保持手绘涂鸦感
- 少部份地方填充黑色让整体压实

人物画法（如需要）：
- 简笔画风格：女孩，短发，圆润的头、点或线表示五官
- 肢体夸张、动作松弛自然
- 可以是完整人物或只露出局部（手、上半身）
```

## 风格后缀（每张图必须完整包含）

```
禁止事项（严格执行）：
- 不要彩色渐变或复杂配色
- 不要粗黑边框或生硬分隔线
- 不要3D效果、阴影、立体感
- 不要密集文字堆砌，文字内容简化
- 保持大量留白和呼吸感
- 装饰元素要克制，不要太多

输出规范：
- 图片比例16:9
- 中文标注
- 直接生成，不需要解释
```

## 提示词组装公式

```
{风格前缀}

{内容描述 — 唯一变量区域}

{风格后缀}
```

## 内容描述写作指南

### 好的内容描述特征

1. **视觉化** — 描述具体的画面场景，而非抽象概念
2. **有层次** — 前景/中景/背景，或左中右布局
3. **有数据** — 融入研究发现的关键数字
4. **有动作** — 人物在做什么，而非静态站立
5. **有标注** — 指定中文文字内容，但保持精简

### 不同图片类型的内容描述模板

**封面图:**
```
画面中央大标题"{主题名}"，旁边一个{人物动作描述}。
周围散落着与主题相关的{图标元素}。
底部小字"{副标题/slogan}"。
```

**数据图:**
```
一个简笔画{图表类型}展示{数据内容}。
旁边一个人物{与数据互动的动作}。
标注关键数字：{核心数据点}。
```

**对比图:**
```
画面分为左右两部分，左边是{旧状态/问题}，右边是{新状态/方案}。
中间用箭头或分隔表示转变。
{人物在两边的不同表情/动作}。
```

**流程图:**
```
从左到右/从上到下的{N}个步骤，用箭头连接。
每个步骤用一个小图标或人物动作表示。
标注每个步骤的关键词。
```

**总结/CTA图:**
```
一个自信的人物站在{成果/目标}旁边，做出{胜利/邀请}的手势。
周围环绕{核心要点}的关键词。
底部"{行动号召语}"。
```

## 多图一致性保障

### 硬性约束（不可修改）

| 约束项 | 规范 |
|-------|------|
| 风格前缀 | 每张完整包含，一字不改 |
| 风格后缀 | 每张完整包含，一字不改 |
| 主色调 | 纯黄色 + 黑色压实 |
| 线条 | 马克笔质感，略带抖动 |
| 人物 | 短发女孩，圆头，点/线五官 |
| 比例 | 16:9 |
| 语言 | 中文标注 |

### 内容差异化

仅通过 [内容描述] 部分体现每张图的独特信息。
不要在风格描述中加入额外的颜色或效果。
