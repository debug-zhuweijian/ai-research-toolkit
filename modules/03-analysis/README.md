# Phase 03: 论文分析与深度研究

论文审阅、深度研究、创意生成与内容分析。

## 包含组件

### Skills（9）

| Skill | 说明 |
|-------|------|
| `paper-review` | 论文结构与质量审阅 |
| `paper-proofread` | 论文校对与语言修正 |
| `academic-paper-reviewer` | 学术论文审稿意见生成 |
| `confidence-check` | 论文结论可信度评估 |
| `deep-research-v5` | 深度研究 v5（多轮迭代式调研） |
| `brainstorming-research-ideas` | 研究创意头脑风暴 |
| `creative-thinking-for-research` | 创造性研究思维拓展 |
| `content-research-writer` | 基于研究内容撰写 |
| `research-ideation` | 研究选题与方向构思 |

### Agents（5）

| Agent | 说明 |
|-------|------|
| `deep-research-agent` | 深度研究执行代理 |
| `research-analyst` | 研究数据分析代理 |
| `bibliography-auditor` | 参考文献审计代理 |
| `technical-reviewer` | 技术细节审查代理 |
| `literature-reviewer` | 文献综述审阅代理 |

## 快速开始

```bash
# 论文审阅
/paper-review

# 深度研究
/deep-research-v5 "EEG speech decoding survey 2024"

# 研究创意生成
/brainstorming-research-ideas
```

## 组件关系

`brainstorming-research-ideas` 与 `creative-thinking-for-research` 互补：
- 前者侧重**研究方向的广度发散**
- 后者侧重**单个问题的深度创造性思考**
- 建议先用 brainstorming 生成候选方向，再用 creative-thinking 深化
