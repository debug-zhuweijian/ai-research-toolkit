# DeepScientist Agents

> **注意**: 这些 agent 需要 [DeepScientist](https://github.com/DoriRoth/DeepScientist) 平台。标准 Claude Code 环境不可用。

## 安装前提

1. 安装 DeepScientist: `pip install deepscientist`
2. 配置 API 密钥
3. 确认平台运行正常

## 14 个 Agent

这些 agent 实现了完整的科研自动化管道：idea → experiment → analysis → write → review → rebuttal。

### 使用流程

1. `deepscientist-idea` — 生成研究想法
2. `deepscientist-scout` — 文献侦察
3. `deepscientist-intake-audit` — 入口审计
4. `deepscientist-baseline` — 基线建立
5. `deepscientist-experiment` — 实验执行
6. `deepscientist-optimize` — 优化迭代
7. `deepscientist-analysis-campaign` — 分析活动
8. `deepscientist-figure-polish` — 图表润色
9. `deepscientist-decision` — 决策支持
10. `deepscientist-write` — 论文撰写
11. `deepscientist-mentor` — 指导审阅
12. `deepscientist-review` — 审阅模拟
13. `deepscientist-rebuttal` — Rebuttal 辅助
14. `deepscientist-finalize` — 最终定稿
