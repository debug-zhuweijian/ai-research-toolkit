# Recommended Obsidian Plugins

## Essential for Research Workflow

| Plugin | Purpose | Install |
|--------|---------|---------|
| **Claudian** | Claude + Obsidian integration, AI chat in vault | Community Plugins → search "Claudian" |
| **Dataview** | Query your notes like a database | Community Plugins → search "Dataview" |
| **Templater** | Advanced templates for paper notes | Community Plugins → search "Templater" |
| **Excalidraw** | Diagram drawing within Obsidian | Community Plugins → search "Excalidraw" |

## Optional but Useful

| Plugin | Purpose |
|--------|---------|
| **Pandoc Plugin** | Export to DOCX/PDF/LaTeX |
| **Citations** | Zotero integration, insert citations |
| **Zotero Integration** | Import Zotero highlights and notes |
| **Calendar** | Daily notes calendar view |
| **Graph Analysis** | Enhanced graph view for knowledge exploration |

## How to Install Community Plugins

1. Open Obsidian → Settings → Community Plugins
2. Turn off "Safe Mode"
3. Click "Browse" → search plugin name → "Install" → "Enable"

## Claudian Setup

Claudian integrates Claude directly into Obsidian:
1. Install Claudian from Community Plugins
2. Configure with your Anthropic API key
3. Select your preferred model
4. Use `Ctrl+Shift+C` to open Claudian chat

This allows you to chat with Claude about your notes, summarize papers, and generate content directly within your vault.
